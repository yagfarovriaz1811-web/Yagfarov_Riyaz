# Weather Diary (Дневник погоды)

## Автор
Адилавов Арсен

## Описание
Weather Diary — это простое GUI-приложение на Python, предназначенное для ведения дневника наблюдений за погодой. 
Приложение позволяет пользователю сохранять записи о дате, температуре, общем описании погоды и наличии осадков.

### Основные возможности:
- **Добавление записей**: Ввод даты, температуры, описания и отметки об осадках.
- **Просмотр**: Все записи отображаются в удобной таблице.
- **Фильтрация**: Возможность фильтровать записи по конкретной дате или минимальной температуре.
- **Удаление**: Можно удалить любую неверную или ненужную запись.
- **Сохранение**: Данные автоматически сохраняются в файл формата JSON и загружаются при следующем запуске.

## Требования
- Python 3.6 или выше
- Библиотека `tkinter` (обычно входит в стандартную поставку Python)
